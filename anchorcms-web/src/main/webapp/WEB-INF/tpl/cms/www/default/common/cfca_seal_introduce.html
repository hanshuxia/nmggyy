电子签章

功能介绍：

电子签章功能是内蒙古工业云为平台企业提供的一种合同在线签署功能，企业采用电子签章签署的合同具有与纸质合同同等的法律效力，同时具有方便查阅、保存、传递等优点。在内蒙古工业云开通电子签章功能后，即可与其他内蒙古工业云注册企业进行在线签署合同。

申请标准：

1.您已通过内蒙古工业云企业实名认证；
2.所需信息包含：企业名称、办公电话、联系人姓名、联系人手机号、电子邮箱、法人代表名称、法人代表身份证号、统一社会信用代码（或工商注册号、税务登记号、组织机构代码）。

收费标准：
