内蒙古航天云网简介
INNER MOMGOLIA CASI CLOUD

内蒙古航联科技开发有限责任公司（以下简称航联公司）成立于1992年，隶属于中国航天科工集团航天云网科技发展有限责任公司（以下简称航天云网公司），是航天云网公司的全资子公司，是航天云网公司在内蒙古落地项目的主要承担及实施单位。

航联公司目前主要负责航天云网工业品共享中心及内蒙古工业云的建设、运营和推广，航天云网客户服务中心及数据中心的管理和运营。工业品共享中心是航天云网三大主要版块之一，是助力工业企业实现转型发展的综合服务云商务平台；内蒙古工业云是航天云网与自治区特色经济相融合的工业互联网平台，是将航天云网服务落地内蒙古、引领自治区优势特色产业向国内外发展的重要载体。

公司成立以来得到了内蒙古自治区党委和政府领导人的高度重视，自治区原党委书记王君、自治区副主席白向群亲临航联公司参观并指导工作，并对航联公司的工作给予了充分肯定。工业品共享中心的上线运营及内蒙古工业云的建设获得了各界认可：自治区经信委将航联公司的平台作为全区“两化融合”重点项目，并向工信部推荐申报互联网与工业融合创新试点单位；自治区国防科工办将平台作为军民融合重点推进项目给予政策支持；自治区科技厅将航联公司的平台作为全区重点自主创新项目写入“十三五规划”，航联公司申报的项目被科技厅、科技局作为科技创新企业列入项目储备并给予相关支持；呼和浩特市商务局拟将平台作为呼和浩特生产型服务业重点打造项目，给予专项资金及政策支持。此外，航天云网公司已与内蒙古自治区政府、内蒙古自治区科技厅、呼和浩特市政府、鄂尔多斯伊金霍洛旗政府签署战略合作协议，冀将航联公司作为实施主体，在公共服务大数据、物联网、智能制造、城市智慧产业以及地区信息化建设等领域开展深入合作。

航联公司秉承"航天品质、信誉第一"的经营理念，以严、慎、细、实的工作作风，精心为内蒙古当地政府及各企事业单位提供便捷、高效的互联网服务，希望能与工业企业共生、共赢、共发展，共创全新的商业模式，以“互联网＋工业”之手助推地方经济转型升级。

内蒙古工业云简介
INNER MOMGOLIA INDUSTRIAL CLOUD

航天云网是航天科工集团打造的工业互联网平台，其三大支柱模块为云制造、创新创业及工业品共享中心。航天云网在运营主体平台的同时，实施地域化运营的战略部署，在全国各地区打造航天云网落地项目，现已实现江西航天云网、贵州工业云、四川航天云网的落地，计划2020年实现全国范围落地。内蒙古工业云是航天云网在内蒙古自治区的落地项目。

内蒙古工业云平台积极响应内蒙古自治区“互联网＋”及“中国制造2025”的落地举措，通过互联网手段解决工业研发、生产、销售、售后服务等产业链问题，帮助内蒙古在国内具有产业优势的企业发挥其领先优势。内蒙古工业云作为开放化的企业公共服务平台，能够为企业搭建起一个庞大的社交网络，促进各商家之间的联盟与合作；解决广大中小微企业资金规模不足的限制，提供高端产品“以租代买”的服务形式，帮助企业降低建设运营成本；缩小中小企业信息化的“数字鸿沟”，为中小企业提供咨询服务、技术交流等；通过平台企业数据采集与政务外网数据采集，为企业提供一般商业平台难以取得的市场分析、行业态势、竞品分析等大数据增值服务；促进中小企业转型升级，推进地区经济发展。

内蒙古工业云平台的战略定位是：建立内蒙古自治区经济运行管理主平台。形成覆盖主要产业及重点区域的“互联网＋工业”生态系统，建立起立足内蒙古、辐射西北、面向全国的云端产业智能配套模式。平台的建设目标是：紧紧围绕以上战略定位，成为“政府管理的助手、企业经营的帮手、地方经济发展的推手”。

在以上思想的指导及前期区内各行业调研的基础上，内蒙古工业云平台目前建立五大功能模块：“协同制造、云资源交易中心、供应链协同、公共服务、软件应用”，致力于为区内企业提供精细化、差异化、定制化的优质服务，提升企业信息化水平，推动工业产业转型升级。通过五大板块的运营，实现以下目标：

1.为政府与企业间构建桥梁

企业通过平台了解政府前沿惠补政策、资源支持政策等，企业可在平台及时申请并享受补贴与支持；政府通过企业在平台的应用数据分析结果，了解区内行业发展动态与趋势，加强经济管控。

2.为企业与企业间构建桥梁

企业用户可通过平台与其他企业实现设计研发、外协采购、生产制造等能力与需求对接；可展示及管理企业备品备件、维修资源与能力，实现区域内资源共享，降本增效。

3.为平台与企业间构建桥梁

企业可通过平台及时获知行业最新动态与讯息；可了解区内云计算资源分布状况并及时租赁或购买到所需云资源、软件系统等；可在平台享有3D打印、创新创业、互联网金融等其他综合服务。
